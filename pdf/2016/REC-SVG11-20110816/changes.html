Appendix Q: Changes

This appendix is informative, not normative.

The only changes made to the document since the SVG 1.1 Second Edition Proposed Recommendation were the following:

  • Some additional wording in the CSS 2.0 reference in the References Appendix to indicate that future versions of the SVG specification will reference CSS 2.1 and CSS Fonts Module Level 3.
  • An update to the informative MathML reference from MathML 2.0 to MathML 3.0.
